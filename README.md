# Lab8 - AWS Static Website with Pulumi

This project deploys a static website on AWS using Pulumi Infrastructure as Code.

## Project Structure
- `ppinfra/` - Pulumi infrastructure code
  - `index.ts` - Main infrastructure definition
  - `www/` - Static website files

## Features
- S3 bucket for static website hosting
- CloudFront CDN for content delivery
- Automated deployment with Pulumi

## Getting Started
1. Navigate to `ppinfra` directory
2. Install dependencies: `npm install`
3. Configure AWS credentials
4. Deploy: `pulumi up`
